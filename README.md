# Full-stack сервис агрегации данных: Праздники 2026

Инструмент для автоматического сбора данных с внешних ресурсов, их структурирования и предоставления через различные интерфейсы.

### Ключевые возможности:
Парсер на базе Axios и Cheerio, который собирает данные о праздниках, обходит DOM-структуру сайта и учитывает локализацию дат.
Автоматическая очистка данных и их миграция в реляционную базу данных SQLite.
* Многоканальный интерфейс:
    * Web-приложение: На базе Fastify и EJS с возможностью интерактивного поиска по датам.
    * CLI-утилита: Консольный инструмент на Node.js для быстрого получения данных прямо из терминала.
Реализован REST-эндпоинт для интеграции с другими сервисами.

### Стек технологий:
Node.js, SQLite, Fastify, Axios, Cheerio, EJS.
